# NeveRipe Coaching

Career coaching website for Sean Zwenger. Built as a static single-page site hosted on GitHub Pages.

**Live site:** [dayne-train.github.io/sean-zwenger-coaching](https://dayne-train.github.io/sean-zwenger-coaching/)

## Stack

- HTML / CSS / vanilla JS
- Google Fonts (Bebas Neue + Source Sans 3)
- GitHub Pages with Actions deploy

## Features

- Dark editorial design with gold accent palette
- Animated rotating hero headline
- Slow marquee testimonial carousel
- Full CSS design token system (spacing, type scale, colors, radii, transitions)
- Accessible: skip link, ARIA landmarks, focus-visible styles, semantic markup, reduced motion support
- Responsive: desktop, tablet, mobile breakpoints

## Setup

To swap in the real intake form, replace the `href="https://tally.so"` in `index.html` with the actual form URL.

To update Sean's photo, replace `sean.jpg` in the root directory.

## Development

Open `index.html` in a browser. No build step required.
